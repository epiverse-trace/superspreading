# Constants used in superspreading

`FINITE_INF` is a large finite number used to approximate `Inf`.

`NSIM` is the number of simulations run when generating random samples
or branching process simulation replicates.

## Usage

``` r
FINITE_INF

NSIM
```

## Format

An object of class `numeric` of length 1.

An object of class `numeric` of length 1.
